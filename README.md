## 🧠 Problem Sets Breakdown

### 🔹 Week 0 – Python Basics
Introduction to input/output and string manipulation:
- `einstein.py` – Energy calculation using E = mc²  
- `faces.py` – Converting text to emojis  
- `indoor.py` – String normalization  
- `playback.py` – Formatting text playback  
- `tip.py` – Tip calculator  

---

### 🔹 Week 1 – Conditionals & Functions
Working with logic and structured code:
- `bank.py` – Greeting-based transaction logic  
- `deep.py` – Case-insensitive comparison  
- `extensions.py` – File type recognition  
- `interpreter.py` – Simple math interpreter  
- `meal.py` – Time-based meal classification  

---

### 🔹 Week 2 – Loops & Data Processing
Handling iteration and structured input:
- `camel.py` – Convert camelCase to snake_case  
- `coke.py` – Vending machine simulation  
- `nutrition.py` – Calorie lookup system  
- `plates.py` – License plate validation  
- `twttr.py` – Text filtering (removing vowels)  

---

### 🔹 Week 3 – Algorithms & Lists
Problem-solving with collections:
- `fuel.py` – Fuel gauge percentage system  
- `grocery.py` – Aggregation and counting  
- `outdated.py` – Date format conversion  
- `taqueria.py` – Menu-based ordering system  

---

### 🔹 Week 4 – Advanced Concepts
Working with libraries and more complex logic:
- `adieu.py` – Dynamic list formatting  
- `emojize.py` – Emoji conversion using a library  
- `figlet.py` – ASCII art text generation  

---

- Python programming fundamentals  
- Input validation and error handling  
- String manipulation and formatting  
- Conditional logic and control flow
